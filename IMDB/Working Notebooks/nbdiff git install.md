# Jupyter Notebook Version Control

Documentation for [nbdiff](https://nbdime.readthedocs.io/en/latest/)

In a command terminal, enter the following commands to integrate nbdiff with git:

```bash
pip install --upgrade nbdime
nbdime config-git --enable --global
```
